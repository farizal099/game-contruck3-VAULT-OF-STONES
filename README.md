🧩 Puzzle Game - Contruck 3

🎮 Deskripsi

Game ini adalah game puzzle yang menantang pemain untuk berpikir dan menyelesaikan setiap level dengan strategi yang tepat.
Setiap level memiliki tingkat kesulitan yang berbeda, jadi kamu harus fokus, teliti, dan kreatif untuk menemukan solusinya.

🧠 Gameplay
Selesaikan puzzle di setiap level
Gunakan logika dan strategi untuk menang
Semakin tinggi level, semakin sulit tantangannya

🕹️ Cara Main
Buka game menggunakan Construct 3 atau browser (jika sudah di-export)
Ikuti aturan di setiap level
Pecahkan puzzle untuk lanjut ke level berikutnya
Selesaikan semua level untuk menjadi pemenang

File utama game ada di repository ini

⚠️ Catatan

🚀 Tujuan
